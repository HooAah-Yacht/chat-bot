## 🔍 분석 가능한 문서 형식

### ✅ 작동하는 문서 형식

1. **텍스트 기반 PDF** (95%+ 성공률)
   - 텍스트를 선택/복사할 수 있는 PDF
   - 예: 최신 오너스 매뉴얼, 클래스 규칙서

2. **간단한 레이아웃 PDF** (90%+ 성공률)
   - 단순한 텍스트 배치
   - 예: 클래스 규칙서

### ⚠️ 부분 작동하는 문서 형식

1. **복잡한 레이아웃 PDF** (30-50% 성공률)
   - 브로셔, 화려한 디자인의 매뉴얼
   - 텍스트는 추출되지만 순서가 뒤섞일 수 있음

### ❌ 작동하지 않는 문서 형식

1. **스캔된 이미지 PDF** (0-10% 성공률)
   - 텍스트를 선택할 수 없음
   - **해결 방안**: OCR 기능 추가 필요 (현재 구현됨, Tesseract 설치 필요)

2. **암호화된 PDF** (0% 성공률)
   - 비밀번호로 보호됨

## 💡 주요 발견 사항

1. **클래스 규칙서가 가장 분석하기 좋음**
   - 부품 정보가 체계적으로 정리되어 있음
   - 평균 100개 이상의 부품 추출 가능

2. **오너스 매뉴얼도 효과적**
   - 상세한 부품 정보 포함
   - 평균 150개 이상의 부품 추출 가능

3. **일부 문서에서 오류 발생**
   - `sailArea` 필드가 None인 경우 처리 필요
   - JSON 구조 불일치 문제
